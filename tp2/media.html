Connaissances

Théoriques (quelles sont les notions que vous avez retenues) ?

Dans le cadre de ce cours, nous avons retenu plusieurs notions sur l'éclairage, le poids des images et des vidéos en relation avec le Web. Nous avons également appris les différences et les effets de différentes fonctionnalités sur Adobe Photoshop, tels que la luminosité, le contraste, etc.

Pratiques (quels sont les logiciels que vous avez appris) ?

Pour ce cours, nous avons appris à utiliser Adobe Photoshop pour la retouche d'images et Adobe Premiere pour la modification de vidéos.

Conditionnelles (quels sont les projets que vous avez réalisés) ?

Comme projet dans ce cours, nous avons dû retravailler plusieurs images fournies par le professeur afin de les rendre plus attrayantes et moins lourdes pour le web. Nous avons également dû retravailler une vidéo pour le web en réduisant son poids et en ajustant l'éclairage.
