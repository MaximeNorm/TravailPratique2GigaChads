Connaissances

Théoriques (quelles sont les notions que vous avez retenues) ?

Dans le cadre de ce cours, nous avons appris les différents services d'hébergement disponibles ainsi que les commandes de terminal nécessaires à la mise en place de serveurs.

Pratiques (quels sont les logiciels que vous avez appris) ?

Dans le cadre de ce cours, nous avons appris à utiliser Amazon Web Services (AWS) ainsi que l'invite de commande sous Windows.

Conditionnelles (quels sont les projets que vous avez réalisés) ?

Dans ce cours, nous avons dû créer un tutoriel pour la mise en place d'un serveur web sur Amazon Web Services (AWS) dans le cadre de notre projet final de cours. Les autres projets étaient des éléments constitutifs de ce tutoriel que nous avons réalisés au préalable. Nous avons également passé un examen théorique final qui comportait des questions sur ce même tutoriel.
